Politique de Confidentialité

Dernière mise à jour : 27 janvier 2025

1. Identité de l'Éditeur

AYNA est une application mobile développée pour améliorer votre pratique spirituelle musulmane.

Contact : pro.ibrahima00@gmail.com

2. Données Collectées

Avec votre consentement explicite, nous collectons des données d'utilisation anonymes et limitées pour améliorer l'application :

  • Fonctionnalités utilisées : Quelles fonctionnalités de l'application vous utilisez (dhikr, journal, Quran, etc.)
  • Navigation : Les écrans visités dans l'application
  • Informations techniques générales : Type d'appareil, version de l'application, système d'exploitation
  • Métriques d'utilisation : Nombre de sessions, durée d'utilisation (agrégées, anonymisées)

3. Données NON Collectées

Nous ne collectons JAMAIS les données suivantes :

  • Contenu de votre journal : Vos notes, pensées, ou écrits personnels
  • Vos messages : Contenu de vos conversations avec l'assistant IA
  • Intentions spirituelles : Vos intentions (niyyah) ou pratiques spirituelles personnelles
  • Localisation précise : Nous ne collectons pas votre position GPS exacte
  • Données personnelles sensibles : Email, nom, mot de passe, ou autres informations d'identification
  • Contenu religieux personnel : Vos pratiques de dhikr, prières, ou autres activités spirituelles individuelles

4. Base Légale

La collecte de données analytiques est basée sur votre consentement explicite (opt-in).

Vous devez activer explicitement les statistiques dans l'application. Sans votre consentement, aucune donnée analytique n'est collectée.

5. Durée de Conservation

Les données analytiques sont conservées pendant 7 jours maximum en local sur votre appareil.

Les événements expirés sont automatiquement supprimés. Les données envoyées à nos serveurs sont conservées conformément à notre politique de rétention (maximum 12 mois).

6. Vos Droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : Vous pouvez demander à voir toutes les données que nous avons collectées vous concernant
  • Droit de suppression : Vous pouvez demander la suppression de toutes vos données analytiques
  • Droit de retrait du consentement : Vous pouvez désactiver les statistiques à tout moment dans les paramètres
  • Droit d'export : Vous pouvez demander une copie de vos données au format JSON

Pour exercer ces droits, contactez-nous à : pro.ibrahima00@gmail.com

7. Transfert de Données

Les données analytiques sont stockées sur des serveurs sécurisés via Supabase (hébergement conforme RGPD).

Aucune donnée n'est transférée à des tiers à des fins publicitaires ou commerciales.

8. Sécurité

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données :

  • Chiffrement des données en transit (HTTPS)
  • Stockage sécurisé avec Row Level Security (RLS)
  • Aucune collecte de données personnelles identifiables (PII)
  • Anonymisation automatique des identifiants utilisateur

9. Cookies et Technologies Similaires

L'application mobile AYNA n'utilise pas de cookies. Nous utilisons uniquement le stockage local de l'appareil (AsyncStorage) pour :

  • Sauvegarder vos préférences utilisateur
  • Permettre le fonctionnement hors ligne
  • Gérer la queue d'événements analytiques (avec votre consentement)

10. Modifications

Nous nous réservons le droit de modifier cette politique de confidentialité. Toute modification sera communiquée dans l'application et la date de mise à jour sera modifiée.

En continuant à utiliser l'application après les modifications, vous acceptez la nouvelle politique.

11. Contact

Pour toute question concernant cette politique de confidentialité ou vos données personnelles, contactez-nous :

Email : pro.ibrahima00@gmail.com
